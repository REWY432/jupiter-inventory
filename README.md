# ООО "Юпитер" - Система учета продукции

Современная веб-система для учета масок и катушек с интеграцией Firebase.

## Возможности

### Маски
- Учет масок фехтовальных (Рапира, Шпага, Сабля, Тренер)
- Умная генерация серийных номеров
- Справочник поколений
- Массовое добавление
- Печать QR-кодов и этикеток
- Буфер отгрузки

### Катушки
- Учет катушек-сматывателей
- Поддержка разных типов проводов (Китайский, Favero)
- Печать паспортов с EAN-13 штрихкодами
- Настройка моделей и годов
- Массовое добавление

### Общее
- Темная/светлая тема
- Адаптивный дизайн (мобильные и десктоп)
- Offline-режим (IndexedDB persistence)
- QR-сканер для быстрого поиска
- Экспорт/импорт CSV
- Аналитика и статистика

## Технологии

- **Vue 3** - современный реактивный фреймворк
- **Vite** - быстрая сборка
- **Pinia** - state management
- **TailwindCSS** - utility-first CSS
- **Firebase** - backend (Firestore + Auth)
- **Chart.js** - графики
- **jsPDF** - генерация PDF

## Структура проекта

\`\`\`
src/
├── assets/          # Стили и ресурсы
├── components/      # Переиспользуемые компоненты
│   ├── common/      # Общие компоненты (модалки, кнопки)
│   ├── masks/       # Компоненты для масок
│   └── spools/      # Компоненты для катушек
├── composables/     # Композиции (хуки)
├── router/          # Роутинг
├── stores/          # Pinia stores
├── utils/           # Утилиты
└── views/           # Страницы
\`\`\`

## Конфигурация Firebase

Обновите файл \`src/config/firebase.js\` с вашими учетными данными Firebase.
